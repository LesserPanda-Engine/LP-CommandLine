# LesserPanda CLI (WIP)

Commandline tool for [LesserPanda engine](https://github.com/pixelpicosean/lesser-panda)

## Support ES2015 features:

- es2015

## Usage

### Start a dev server

`lpanda server`

### Build production bundle

`lpanda build`
